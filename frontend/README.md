# Bungalov Frontend (Next.js)

## Kurulum
1. `npm install`
2. `.env.example` dosyasini `.env.local` olarak kopyala
3. `npm run dev`

## Deploy
- Frontend: Vercel
- Backend API: Render

## Ozellikler
- URL tabanli i18n: `/tr`, `/en`, `/ar`
- SEO: metadata, `sitemap.xml`, `robots.txt`
- Public + admin sayfalari
